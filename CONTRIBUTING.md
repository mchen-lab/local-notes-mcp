## Project Philosophy

This project is designed to be a **focused, feature-complete utility** for local note-taking with MCP integration. It intentionally maintains a small scope.

**We value your contributions!** However, to keep the project focused, we ask that you **please open an issue to discuss significant changes before submitting a Pull Request**. The most helpful way to contribute right now is by using the app and reporting bugs.

## Code of Conduct

By participating in this project, you agree to maintain a respectful and inclusive environment for everyone.

Bug reports are very valuable. Before creating one, please check existing issues to avoid duplicates.

When creating a bug report, please include:
- **Clear title** describing the issue
- **Steps to reproduce** the behavior
- **Expected behavior** vs actual behavior
- **Screenshots** if applicable
- **Environment details** (OS, Browser, Docker version, etc.)
